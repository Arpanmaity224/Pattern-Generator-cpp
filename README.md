# Pattern Generator in C++

A simple menu-driven C++ project to generate different patterns using loops.

## 🚀 Features
- Square Pattern
- Triangle Pattern
- Floyd’s Triangle
- User choice based menu

## 🛠️ Tech Used
- C++
- Loops & Conditional Statements
